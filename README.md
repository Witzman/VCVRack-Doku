# VCVRack-Doku

Patcher's handbook and module reference for [VCV Rack](https://vcvrack.com/). Built as a knowledge base for AI-assisted patching conversations.

## Reading the docs

Open `Rack-Doku/index.html` in a browser.

## Contents

- `htmldoku/` — Markdown source files
- `Rack-Doku/` — Static HTML documentation site

## Regenerating

Documentation is generated from the VCV Rack source repos using a structured AI prompt.

From the `rack/` workspace root:
```
Read rack-invoke.md and apply
```

See `rack-invoke.md` and `generator.md` in the parent workspace for the full process and configuration.
